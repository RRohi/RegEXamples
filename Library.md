# A list of various RegEx patterns.

## Match everything after the last occurance of _
```regex
(?!.+_)(?<=_).+
```

## IPv4 Address match.
```regex
^([1-9]\d?|1\d{2}|2([0-4]\d|5[0-5]))\.((0|\d{1,2}|1\d{2}|2([0-4]\d|5[0-5]))\.){2}((\d|[1-9]\d?)|1\d{2}|2([0-4]\d|5[0-5]))$
```
### Matches:
```
10.10.101.10
255.255.255.255
192.63.2.102
213.25.2.0
1.0.0.1
```
### Doesn't match:
```
256.1.1.1
255.256.1.1
255.255.256.1
255.255.255.256
0.0.0.0
```
